This module explores the fundamentals of computer networking and network security.

## Learning Goals

By the end of this module you should be able to:

- Understand the fundamentals of computer networking and explain key networking concepts such as network layers and protocols.

- Differentiate between wired and wireless networks and their advantages and disadvantages.

- Use a traffic analyser to capture real network traffic.

- Analyse network traffic using Wireshark on Raspberry Pi and interpret network packets to identify protocols, detect anomalies, and understand data transmission.

- Identify common network security threats, including packet sniffing and man-in-the-middle attacks and discuss the attack prevention and mitigation techniques.
