:::tip[Activity: Think about it!]

- How does the role of the MicroPython interpreter on the Pico 2W differ from native Python on the Raspberry Pi 4B?
- Why might you choose a microcontroller (Pico) over a single-board computer (Pi 4B) for certain projects?
- What are the implications of the Pico's non-persistent storage compared to the Pi 4B's SD card storage?

:::

:::note

The Pico 2W has inbuilt wireless capabilities. This opens up a route for a wide range of activities and projects that can be performed with the Pico 2W. You can:

- Set up the Pico 2W as a basic web server
- Create a remote LED control system using WiFi

:::
